The Purpose of Pomo Progress!

The purpose of this website is to be a tool that helps users achieve goals they set for themselves, whatever goals they want. Goals like losing weight, finishing an art project, or learning a new skill.

Speaking from personal experience, one of the biggest challenges in completing goals, is maintain consistency of effort. We might start of strong and pursue a goal with vigor for a few weeks, but often time our enthusiasm with dissipate, and out progress will fall off. This website will help you track your progress, and let you know whether or not your on pace to achieve your goal.

Inspired By and Based on the Pomodoro Technique™

This site was inspired by the Pomodoro Technique and will make a lot more sense if you're already familiar with it. The learning technique was developed by Francesco Cirillo in the late 1980s. The basics of the technique involve working for a set amount of time--about 25 minutes--and then taking a 3 to 5 minute break. After working a 25 minute block four times in a row, (with short breaks in-between), practitioners are encouraged to take a 15 - 30 minute break before attempting another set. The reasoning for this is because people are supposedly better able to retain the information they just learned a bit better.

The Pomodoro Technique™ is based around the act of learning, but Pomo Progress! is a broad tool that could be applied to many different pursuits. In fact, if you want to eschew the tenets of the aforementioned technique, you can use this web app any way you'd like.

Version 1.0 of this website does not have a built-in timer; its basic purpose is to simply be an organization tool to help you achieve longer-term goals. Use any external timer you want to time intervals. The Google timer is simple and effective. I'm hoping to add an embedded timer in a future version.

Deadlines Help You Achieve Goals

A major tenant that this website is based on is Parkinson's law. In a nutshell, Parkinson's law basically states that a specific task will be completed by the deadline that was set to finish it by. For example, if you have a month to finish a twenty page term paper, it will be completed in a month. If you have two days to write a five page term paper, it will be written in two days. I think any current or former college students can attest to the validity of this principle. As such, this table tries to make use of 'deadline pressure' and asks you to give the total amount of hours you want to achieve a goal by, and then it asks you for a deadline date. As long as the total amount of hours and and deadline is reasonable, there is a very good chance you will complete the task, due to the invisible forces behind Parkinson's law.

Additionally, if you'd prefer not to put in an hour-count and/or deadline, that's totally fine. Unfortunately, some statistics won't be calculated because without those variables, but you can still track your progress and watch it grow.

Creating and Using a Table

How to Create a New Table

Creating a table is easy, simply click the "create new table" button on the load page to trigger a drop down form that is pretty self-explanatory. The only required inputs are table name and the password fields. You can leave the rest of the fields blank if you wish. However, to get the most of out of this website, it is highly recommended that you use all of the use all of the data fields.

  • new table name - the table name basically works like a username. The name must be unique and is linked with the password you choose.
  • password / confirm password - choose a password that's secure, and one you can remember. Currently, there is no password recovery feature, so it's a good idea to write down your password.

Viewing the Data Table

Date-the date of your most recent entry. Multiple entries on the same date will be combined into one entry. Multiple tasks and notes can be added to the same date.

Pomodoros-shows the amount of pomodoros you accumulated for a given date. After 7 days, the daily amounts of pomodoros are added together for a weekly total.

Task(s)-lists the task that you focused on for the day. You can have more than one task for a given day.

Notes-you have the option to add a short note each time you add more pomodoros to the table. please keep the notes as brief as possible, as there is a character limit on notes. A new notes system is being worked out for the next version

If you want to add new notes or tasks without adding pomodoros, you can put a '0' in that field, and then write in the task and/or notes you want to add to that day.

Other Features

Another notable feature is the journal page, which allows you to type out and save your thoughts about your journey without any character limit. This allows you to workout your deeper thoughts on your progress and lets you plan ahead. Journaling and planning out your progress step-by-step should help you realize and make progress towards your goal, especially if the goal is something rather vague and complex, like making a career change over the next year or two.

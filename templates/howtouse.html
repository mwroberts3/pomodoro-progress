How to Use Pomo Progress

{% endblock %} {% block main %}

This site will make a lot more sense if you're already familiar with the Pomodoro Technique, pioneered by [inventor's name], because this site basically acts as a way to track your progress via the Pomodoro Technique. And, the ultimate purpose, is to just provide a tool that will hopefully help you acheive you're goal. Tracking your progress towards a goal will help keep you motivated and ultimately more likely to acheive it. [link to studies showing this]. Pomo Progress lets you add a deadline to a new table (but not required), so you know how many days you have to acheive your goal. There is a principle [look up principle]s that shows how tasks fit into deadlines, and if you set a deadline you're more likely to finish the task by within that certain amount of time.

Everytime you work for the timelimit, you can add a pomodoro to the table for that day, along with a task name, and notes you may want to add. It may be easier to work for several pomodoros and then add them all at once.
